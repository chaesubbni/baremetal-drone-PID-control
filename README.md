# baremetal-drone-PID-control
학교 지원 사업을 통해 수행한 프로젝트로, 외부 라이브러리 없이 레지스터를 직접 제어하여 드론의 수평 유지 시스템을 구현했습니다. 센서 데이터 기반의 **확장 칼만 필터(EKF)**로 정밀한 자세를 추정하고, PID 제어 알고리즘을 통해 모터를 실시간으로 제어합니다.
